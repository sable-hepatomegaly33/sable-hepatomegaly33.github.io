⚙️ Advanced settings

While the software works perfectly with default settings, you can adjust several features:

  • Auto-connect: Enable this option under settings to start the secure tunnel when you turn on your computer.
  • Kill switch: Activate this feature to stop all internet traffic if your secure connection drops for any reason. This prevents data leaks.
  • Protocol selection: The software chooses the best protocol for your connection by default, but you can manually select specific encryption standards in the advanced menu.
  • Split tunneling: Choose specific programs that should bypass the secure tunnel if you need local network access simultaneously.

❓ Frequently asked questions

Does this software slow down my internet speed?

Encryption adds a very small overhead to your data transfer. Most modern internet connections do not notice a difference. Ensure you choose a server close to your actual location for optimal speed.

Can I use this on more than one computer?

Yes, you can install the software on every machine you use. The account credentials work across all supported Windows devices.

Does the software keep logs of my browsing?

We believe in strict privacy. The software does not store your browsing history, bandwidth usage, or connection logs. Your activity stays private.

If you encounter trouble, check these common fixes first:

  • Installer errors: Right-click the installation file and select "Run as administrator" to ensure the installer has the correct permissions to modify your system registry.
